1. git 사용법
1) pwd
=> 현재 나의 위치
2) ls
=> 현재 내 위치(폴더) 내 파일목록 확인
( ls -a : 숨긴폴더까지 확인가능)
3) cd
=> 체인지 디렉토리(현위치변경)
4) mkdir testdir
=> 'testdir' 폴더 형성
5) touch test.txt
=> 'test' 라는 텍스트 파일을 만든다
6) touch .hidden
=> .hidden이라는 확장자를 가진 파일 형성
7) echo 'hello'
=> 'hello' 문자를 출력
8) echo 'hello' >> text.txt 
=> 'hello' 문자를 텍스트 파일에 보냄
9) cat test.txt
=> 텍스트 파일을 읽어서 화면에 보여줌
10) rm text.txt && .hidden
=> 텍스트파일과 히든파일을 삭제
11) rm -r / rmdir testdir / rm -rf testdir (강제제거 옵션)
=> testdir 폴더를 삭제
12) cd ..
=> 상위폴더로 한칸 위 올라감
13) . 현재위치 .. 상위폴더

2. git 로컬 등록
1) git init
=> 현재 폴더를 루트 디렉토리로
* git config --list
=> 환경설정
2) git config --global user.name/email 
=> 이름 이메일 등록(갱신x)
3) git status
=> 현재 로컬 내에 인지하고 있는 폴더
4) git add 파일이름 또는 .
=> 파일을 인지하도록 등록
* git rm --cached a. txt
=> 등록된 파일 해제



3. 원격 로컬 만들기
1) 로컬이랑 이름 맞추기
git remote add origin
