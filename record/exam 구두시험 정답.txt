stringbuilder : string 타입의 값은 여러 값들을 추가하는데에 성능이 좋지 않기 때문에 stringbuilder의 append메서드를
		통해 문자열을 연결한다

urlencoder : 디코딩된 어떤 데이터를 인코딩하기 위한 클래스로 api 요청변수를 인코딩하는 경우에 쓰인다

httpurlconnection : api 서버와의 연결을 생성하는 역할을 한다

bufferedreader : api 서버의 응답데이터를 읽어들일때 사용하는 스트림으로, 내부 버퍼로 인해 입력속도가 높아지며,
		행 단위 입력이 가능한 readline 메서드를 가지고있다

inputstreamreader : api 응답 스트림은 바이트 스트림으로 문자입력에 적절치 않기 때문에, inputstreamreader을 쓰면
 		바이트 입력 스트림을 문자 입력 스트림으로 변환하여, api로부터 문자 데이터를 받도록 처리할 수 있다
