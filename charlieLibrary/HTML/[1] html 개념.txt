[2] html 개념

1. HTML
=> HYPER TEXT MARKUP LANGUAGE
=> 하이퍼 텍스트(링크를 만드는 텍스트)를 만드는 언어


2. 태그의 특징
=> 정해진 태그를 사용
=> <> 안에 작성
=> 소문자로 작성
=> 계층구조로 작성(부모 - 자식)


3. 태그의 종류
1) 자식태그	: <여는태그></닫는태그>
2) 나홀로 태그	: <태그>, <태그/>


4. 태그의 기본 구성
1) html:5 선언 : <!DOCTYPE html>, 문서 타입선언			& html:5
2) <html> 태그 : 전체내용 포함
3) <head> 태그 : 헤더
4) <body> 태그 : 본문


5. vscode 단축키
1) 주석 : ctrl + /
2) 라인 복사 : alt + shift + 방향키
3) 라인 삭제 : ctrl + x, shift + delete
4) 라인 이동 : alt + 방향키
5) 자동완성 : ctrl + space
