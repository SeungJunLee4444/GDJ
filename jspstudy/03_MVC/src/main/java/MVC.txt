2. model2
- 비즈니스 로직과 화면처리를 분리 
- 동작방식
1) 요청처리
2) 비즈니스 로직 / 화면처리
3) DAO
4) DB

- model2의 가장대표적인 패턴이 MVC다

3. MVC
- 정의 : 화면처리(V), 비즈니스로직(M), 요청처리(C)를 분리하여 프로그램을 개발하는 디자인 패턴
- CONTROLLER : SERVLET, 화면처리
- MODEL : 자바클래스, 비즈니스로직
- VIEW : JSP, 화면처리
